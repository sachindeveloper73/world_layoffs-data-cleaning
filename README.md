# World Layoffs Data Cleaning Project

## Overview
This project demonstrates SQL-based data cleaning and analysis on a layoffs dataset.

## Steps Performed
- Data staging
- Duplicate removal using ROW_NUMBER()
- Data standardization
- NULL handling
- Date format conversion

## Results
- Cleaner dataset
- Ready for analysis
- Insights generated using SQL

## Files
- data_cleaning.sql
- analysis.sql
- dataset/layoffs.csv
